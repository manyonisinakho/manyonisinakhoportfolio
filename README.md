# Sinakho Manyoni — Portfolio

Personal portfolio site for Sinakho Manyoni, a data analytics professional based in Johannesburg, South Africa.

🔗 **Live site:** _add your GitHub Pages URL here once deployed_

## About

This is a single-page portfolio built to showcase experience in data analytics, information management, and business intelligence — including work with Excel, SQL, Python/pandas, and Power BI.

Sections include:
- Professional summary
- Experience (Wild Impact, Startek)
- Skills & core competencies
- Certifications & learning (Microsoft DP-700, Google Data Analytics, freeCodeCamp)
- Education (University of Johannesburg, Balmoral College)
- Contact

## Tech

- Plain HTML, CSS, and vanilla JavaScript — no build step, no dependencies
- Fonts: [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk), [IBM Plex Sans](https://fonts.google.com/specimen/IBM+Plex+Sans), and [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono) via Google Fonts
- Fully responsive, keyboard-accessible, and respects `prefers-reduced-motion`

## Running locally

No install required — just open `index.html` in a browser.

For live-reload while editing, you can optionally serve it locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This site is deployed via **GitHub Pages**:

1. Push changes to the `main` branch
2. GitHub Pages rebuilds automatically from **Settings → Pages**
3. Changes go live within a minute or two

## Updating content

All content lives in `index.html` — edit the relevant section (Experience, Skills, Certifications, etc.) directly and commit. No templating or data files involved.
